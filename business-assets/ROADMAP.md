# Chatlippytm.ai.Bots Business Asset Roadmap

## Role

`Chatlippytm.ai.Bots` is the AI bot workflow hub for lead capture, FAQs, qualification, routing, follow-up, ManyChat/BotBuilders planning, and human handoff.

## Primary CTA

**Build a Bot Workflow**

## Website destination

`/roadmap#bot-workflow`

## CRM tag

`Bot Workflow`

## Roadmap phases

### Phase 1: Define the First Bot Job

Pick one clear job: lead capture, FAQ, appointment routing, affiliate routing, support triage, or follow-up.

### Phase 2: Map the Conversation

Document the first conversation path from visitor question to next action.

### Phase 3: Create Canva Bot Workflow Assets

Create bot workflow cards, conversation maps, service graphics, and social carousels.

### Phase 4: Publish the Bot Workflow Website Section

Connect the Canva card to `/roadmap#bot-workflow` or a dedicated bot workflow page.

### Phase 5: Add CRM + Tags

Track bot workflow interest, lead source, business type, and requested bot function.

### Phase 6: Connect to ManyChat / BotBuilders Concepts

Use the workflow map to guide future ManyChat, BotBuilders, or website-chat implementation.

### Phase 7: Monetize Bot Workflow Services

Offer bot flow maps, implementation packages, chatbot setup, and automation retainers.

## Best practices

- Start with one bot job.
- Keep the first workflow simple.
- Always include human handoff.
- Track the lead source.
- Avoid claiming bots replace all human work.
- Use GitHub as the source of truth for bot scripts and workflow maps.
