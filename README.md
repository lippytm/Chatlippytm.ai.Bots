## 🚀 Getting Started

1. **Clone this hub:**
   ```bash
   git clone https://github.com/lippytm/Chatlippytm.ai.Bots.git
   cd Chatlippytm.ai.Bots
   ```

2. **Run the bot connector** to verify all repository connections:
   ```bash
   python bot_hub.py
   ```

3. **Explore the full catalog** in [`connections.json`](./connections.json).
