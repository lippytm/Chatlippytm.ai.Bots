# Project Role

## Purpose
`Chatlippytm.ai.Bots` is the conversational AI interface hub for the wider lippytm ecosystem.

## Primary Objectives
- define chatbot identity and interaction patterns
- document prompt and conversation design ideas
- support public-facing assistant concepts and demos
- connect chat experiences to the broader ecosystem of products, education, and internal AI ops

## Inputs
- conversation ideas
- prompt designs
- UX patterns
- public assistant use cases
- ecosystem integration needs

## Outputs
- chat interface documentation
- prompt and persona notes
- conversational architecture ideas
- demo and use-case guidance

## Connected Repositories
- `lippytm.ai`
- `Clawlippytm.ai.Bots`
- `MyClaw.lippytm.AI-`
- `lippytm-lippytm.ai-tower-control-ai`

## KPIs
- clarity of assistant identity
- usability of conversation patterns
- integration quality with product and ops hubs
- usefulness for AI learning and experimentation
